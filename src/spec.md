# Chicken Banana Landing Page

## Overview
A minimal landing page that displays only the text "Chicken Banana" centered on the page.

## Features
- Display the text "Chicken Banana" centered both vertically and horizontally
- Plain white background
- Black text in a clean, readable font
- Responsive design that works on different screen sizes

## Technical Requirements
- No backend functionality required
- Static frontend-only application
- No data persistence needed
- No user interactions required
